# Poker Lab

Poker Lab is a lightweight interactive website for learning practical poker strategy.

## Features

- Strategy roadmap covering preflop, position, postflop, poker math, tournaments, and exploitative play
- Interactive strategy trainer for common preflop spots
- Visual 13x13 opening hand matrix
- Quick multiple-choice quiz for self-testing

## Run locally

Because this project is a static site, you can serve it with any simple HTTP server.

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000> in your browser.
